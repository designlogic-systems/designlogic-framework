## Purpose

The examples directory illustrates how DesignLogic Framework surfaces may be combined in bounded, public-facing scenarios. Examples are explanatory aids for navigating framework concepts; they are not implementations, validated products, operational workflows, or evidence of adoption.

```text
DesignLogic Semantic Runtime Architecture
= public architecture layer

DesignLogic Framework
= applied reusable modules built from the architecture

DesignLogic Apps
= products and capabilities built from the framework
```
